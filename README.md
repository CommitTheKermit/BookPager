# 오픈미션 - 페이져 (Pager)

> 실물 책을 액자에 넣어서 한장씩 넘겨서 동적인 액자 구성, 아날로그의 감성을 챙기되 디지털과 결합
>

## 프로젝트 계기

<aside>

최근에 내셔널지오그래픽 잡지를 읽고 있는데, 사진 위주의 잡지라 액자에 걸어두면 참 좋겠다는 생각이 들었습니다. 그런데, 그렇게 하면 전체 책에서 펼치고 있는 페이지만 볼 수 있으니까, 자동으로 넘겨서 모든 페이지를 볼 수 있으면 좋겠다는 생각이 들었습니다.

</aside>

## 프로젝트 목표

- 실물 책을 액자에 넣고 자동으로 페이지 넘기기
- 안드로이드 앱으로 원격 제어
- 3D 프린팅, 3D 모델링, IOT 제어, 안드로이드 앱(Kotlin, Compose)
